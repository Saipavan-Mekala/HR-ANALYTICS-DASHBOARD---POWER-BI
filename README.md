# HR-ANALYTICS-DASHBOARD---POWER-BI

<b> OVERVIEW OF THIS PROJECT</b>

Help an organization to improve employee performance retention (reduce attrition) by creating a HR ANALYTICS DASHBOARD

<b>Description About the Dataset</b>

The dataset consists of 14710 observations and 8 variables. Each row in dataset represents an employee; each column contains employee attributes:

Independent Variables were:

1. Age: Age of employees
2. Department: Department of work
3. Distance from home
4. Education: 1-Below College; 2-College; 3-Bachelor; 4-Master; 5-Doctor
5. Education Field
6. Environment Satisfaction: 1-Low; 2-Medium; 3-High; 4-Very High
7. Job Satisfaction: 1-Low; 2-Medium; 3-High; 4-Very High
8. Marital Status
9. Monthly Income
10. Num Companies Worked: Number of companies worked
11. Work Life Balance: 1-Bad; 2-Good; 3-Better; 4-Best
12. Years At Company: Current years of service
13. Dependent Variable was Attrition: Employee attrition status(0 or 1)

<b>Project Learnings</b>
1. Identified Key Factors to reduce attrition
2. Improved the hiring process
3. Imroved Employee Experience
4. Made workforce more productive
5. Gained employee trust
